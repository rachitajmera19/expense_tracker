Expense Tracker 

A clean, minimal personal finance tracker built entirely with vanilla HTML, CSS, and JavaScript. No backend, no frameworks — just fast, functional, and works offline.

Features
- Add income and expense transactions with category and date
- Visual spending breakdown by category (bar chart)
- Filter transactions by category
- Data persists locally (browser localStorage — no account needed)
- Fully responsive, works on mobile

Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Storage:** Browser LocalStorage (no backend required)

How to Run
1. Clone this repo: `git clone https://github.com/rachitajmera19/expense_tracker`
2. Open `index.html` directly in any browser — that's it!

What I Learned
- DOM manipulation and event-driven programming in JavaScript
- Working with browser LocalStorage for persistent data
- Dynamic chart rendering without any library
- Building clean, responsive UIs with CSS Grid and Flexbox
- Real-world application logic (filtering, aggregation, state management)
